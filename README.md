# utonic-contracts

### clone the repo

```
$ git clone git@github.com:UTONICFinance/utonic-contracts.git
```

### dependencies

```
$ cd utonic-contracts
$ npm install
```

### compile

To compile minter

```
$ npm run compileminter
```

To compile wallet

```
$ npm run compilewallet
```

To compile withdraw

```
$ npm run compilewithdraw
```
